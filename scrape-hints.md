Am Ende jedes article für ein AMV gibt es verschiedene downloadoptionen, z.B:
Download AMV [Torrent] 86.54 Mb 1280x720@29.971fps

das für uns wichtige ist ist der Link zum .torrent file.
Es kann passieren, dass es mehre gibt, Beispiel:
Download AMV [Torrent] 140.99 Mb 1920x1080@25fps
Download AMV [Torrent] 159.64 Mb 1920x1080@25fps
Download preview 30.58 Mb 640x360@25fps 

in diesem fall nehmen wir heuristisch einfach an, dass das größte File die beste Qualität
liefert und laden das torrent file dafür herunter.
Ein deeplink zum .torrent kann z.B. so
https://amvnews.ru/index.php?go=Files&file=downtorrent&id=12671&alt=1
oder so:
https://amvnews.ru/index.php?go=Files&file=downtorrent&id=12671&alt=0

aussehen.
